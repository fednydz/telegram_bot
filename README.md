# 🎬 telegram_bot
بوت تلغرام يقوم بتقسيم الفيديوهات المرسلة إلى أجزاء مدة كل جزء **30 ثانية**.

## 🚀 الميزات
- تقسيم تلقائي وسريع
- معالجة غير متزامنة (لا يعلق البوت)
- تنظيف تلقائي للملفات المؤقتة
- متوافق مع Railway & GitHub

## 🛠 التشغيل المحلي
```bash
pip install -r requirements.txt
cp .env.example .env
# عدل .env وأضف التوكن
